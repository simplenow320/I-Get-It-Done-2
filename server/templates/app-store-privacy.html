App Store Connect Privacy Disclosure

Information for completing the App Privacy section in App Store Connect

Quick Summary: I GET IT DONE collects email, optional display name, audio (processed only - not stored), usage data, device identifiers, and push notification tokens. We do NOT use data for tracking or share with data brokers.

Data Types Collected

When filling out App Store Connect's privacy questionnaire, select the following:

Contact Info
Email Address Yes - Required for account
Name Optional - Display name
User Content
Audio Data Voice recordings (processed, not stored)
Other User Content Tasks, notes, subtasks
Identifiers
User ID Yes - Account identifier
Device ID Yes - For device management
Usage Data
Product Interaction Feature usage, streaks, achievements
Purchases
Purchase History Via App Store / Stripe (subscription status)

Data Use Purposes

For each data type, select these purposes:

Data Type Purpose
Email Address App Functionality, Account Management
Name App Functionality (personalization)
Audio Data App Functionality (voice-to-text for task capture)
User Content App Functionality
User ID App Functionality, Account Management
Device ID App Functionality (push notifications)
Product Interaction App Functionality (gamification features)
Purchase History App Functionality (subscription access)

Is this data used for tracking?

Answer: NO - We do NOT use any collected data for tracking purposes. We do not share data with data brokers, advertising networks, or use it to target ads.

Third-Party SDKs

Disclose these third-party integrations:

  • Expo SDK: App framework (analytics disabled)
  • Stripe: Payment processing (for non-iOS payments)
  • OpenAI API: AI task extraction (server-side only, no SDK)
  • Deepgram API: Voice transcription (server-side only, no SDK)
